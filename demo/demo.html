Less Color Utility Help Hide Help

I'm tired of using Less Color Functions to create related colors by having to edit my less file, save it, wait for Tomcat to auto reload it, invalidate the WRO4J cache and reload my page. This is WAY TOO SLOW when I'm completely guessing at things like the amount to fade a color. So I created this little utility that gives you all the Less Color Functions which you can run and get the results immediately. The selected color function is run when any of the inputs is changed (requires tabbing off the input or pressing enter). Enjoy!
